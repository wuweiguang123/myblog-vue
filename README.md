# myblog-vue

本仓库是myblog的前端部分。后端项目地址为：https://github.com/wuweiguang123/myblog

前端技术栈：Vue+VueRouter+Vuex+axios+ElementUI

项目中使用了markdown-it插件将markdown语法解析为html标签进行渲染显示。

使用mavon-editor插件编辑文章

使用github-markdown-css美化渲染后的文章内容页面

## 安装依赖

```
npm install
```

### 编译启动
```
npm run serve
```

### 构建生产
```
npm run build
```
